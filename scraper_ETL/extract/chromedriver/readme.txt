main.py needs chromedriver file in this folder.
Please download the version of chromedriver file that correspond to the Google Chrome browser version and the architecture of the PC using main.py. Unzip the .zip file

https://chromedriver.chromium.org/

Example:
Download file "ChromeDriver 102.0.5005.61" for MacOS 64bits from...
  https://chromedriver.storage.googleapis.com/index.html?path=102.0.5005.61/
for Google Chrome browser Version 102.0.5005.61
